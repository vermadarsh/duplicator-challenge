A simple plugin that scans the site root folder and saves the entries into the database.

== Description ==

This is a simple plugin, part of a practical test asked to be performed by Awesome Motive. This plugin helps the administrators to scan the site.

= Special Features =
* Site scanner.
* Scan saved into the database to future references.

== Installation ==
1. Upload the entire `duplicator-challenge` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

== Frequently Asked Questions ==

= Can this plugin scan a 10 gigs site? =

Yes, the scanning is performed in chunks.

= Is there any external services/libraries used to do the scan? =

No, the plugin uses core PHP and WordPress functions.
